# VS Protocol Web Interface - 项目总结

## 📋 项目概述

## 🏗️ 技术架构

### 后端技术栈
- **Flask 3.0.0**: Web框架
- **Flask-SocketIO 5.3.5**: WebSocket支持
- **PyYAML 6.0.1**: YAML配置文件处理
- **Python subprocess**: 运行虚拟筛选流程
- **Threading**: 后台任务处理

### 前端技术栈
- **原生HTML5**: 语义化标签
- **原生CSS3**: 现代化样式，响应式设计
- **原生JavaScript (ES6+)**: 无需额外框架
- **Socket.IO Client**: WebSocket客户端
- **Flexbox & Grid**: 布局系统

### 通信架构
```
┌─────────────┐    HTTP/WebSocket    ┌──────────────┐    subprocess    ┌──────────────────┐
```

## 📊 功能模块

### 10个虚拟筛选模块

1. **理化性质筛选** (physicochemical)
   - MW, LogP, nHA, nHD, nRot, nRing, TPSA, QED, SAScore

2. **ADMET筛选** (admet)
   - HIA, F(20%), F(30%), VD, CL, Caco-2, hERG, H-HT, Ames

3. **受体准备** (receptor_prep)
   - PDB清理，配体提取，PDBQT转换

4. **已知抑制剂过滤** (known_inhibitors)
   - 子结构过滤
   - 相似性过滤
   - 药物相似性预测

5. **分子对接** (docking)
   - Uni-Dock对接
   - 氢键识别
   - 关键残基过滤

6. **聚类分析** (clustering)
   - Morgan指纹生成
   - Taylor-Butina聚类

7. **MD前处理** (pre_md)
   - HBA计算
   - 脱靶预测

8. **脱靶预测** (off_target)
   - 多靶点对接

9. **加权评分** (weighted)
   - 多指标综合评分

10. **分子动力学** (md)
    - MD模拟任务提交

## 📝 使用流程

### 标准工作流程

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动服务**
   ```bash
   ./start_web.sh
   ```

3. **访问界面**
   ```
   http://localhost:5000
   ```

4. **选择配置**
   - 在下拉菜单选择配置文件
   - 编辑配置（可选）
   - 保存更改（可选）

5. **启动流程**
   - 点击"启动流程"按钮
   - 监控进度和日志

6. **查看结果**
   - 流程完成后查看输出
   - 导出日志（可选）
